#18600 datalab

Posted on 2016-09-04   |   In Tech   |     |   visitors

18-600 : Foundations of Computer Systems 是CMU ECE系在2016Fall为了解决15213课容量太小太多人选而新开的一门课，其内容包括了以往的CS系的“镇校神课”15-513和15/18-213(Introduction to Computer Systems，也叫ICS或CSAPP)，还加上了ECE 18-640(Foundation of Computer Architectures)的一部分内容。“镇校神课“15213是CMU计算机系本科生必修的一门基础课，也是ECE和CS很多其它分支课程的prerequisite，以课程知识量涵盖广和workload重而闻名。
以前本科计算机组成原理水水过了，也没有选过编译原理，总感觉知识结构里缺了很重要的一环，所以想选这门课打打基础。

第一次的lab叫datalab，一共包含13题。跟往年CSAPP的datalab有几题重复，也有几题不一样，但要求都是利用bit operation做的一些操作，前10题与整形数integer相关，后3题与单精度浮点数float相关。因为要训练对位操作的理解，所以每一题对解题所用到的操作符数量和操作符的种类（’=’不算入操作符，可以随便用）都有比较严格的限制。而且，也要求尽量不要用到大小超过0x7f的写死的数字来帮助运算。

Leetcode 44: Wildcard Matching

Posted on 2016-08-23   |   In Tech   |     |   visitors

题目如下：

Implement wildcard pattern matching with support for ‘?’ and ‘‘.
‘?’ Matches any single character.
‘‘ Matches any sequence of characters (including the empty sequence).

The matching should cover the entire input string (not partial).

The function prototype should be:
bool isMatch(const char s, const char p)

Some examples:
isMatch(“aa”,”a”) → false
isMatch(“aa”,”aa”) → true
isMatch(“aaa”,”aa”) → false
isMatch(“aa”, ““) → true
isMatch(“aa”, “a“) → true
isMatch(“ab”, “?“) → true
isMatch(“aab”, “ca*b”) → false

iOS-利用Catogory给已存在对象添加实例变量

Posted on 2016-03-24   |   In Tech   |     |   visitors

之前线上笔试被问到这道题，之前在实践的时候由于要实现需求，做过类似的事情，现在将其整理一下。

Category与实例变量

Category是Objective-C中的一个扩展机制，通过Category，开发者可以在外部给一个类添加新的方法，从而达到扩展已知类的作用，这是比继承要更加方便的扩展方法，有助于把方法分类应用到不同的需求中，此外也可以帮助开发者对不开源的库进行自己的调整。针对后一种情况，开发者很有可能会需要添加自己想要的实例变量，但是，在编译时，Objective-C是不允许向Category添加实例变量的（虽然允许声明属性，但不会自动生成getter/setter），但是也并非没有办法。OC的运行时特性可以实现这样的需求。
